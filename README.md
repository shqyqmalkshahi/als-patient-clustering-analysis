# ALS Patient Clustering Analysis

## Overview
This project applies unsupervised machine learning techniques to cluster ALS patients based on clinical features.

## Objective
To identify meaningful patient groupings using K-Means clustering and visualize patterns using PCA.

## Methods
- Data cleaning and preprocessing
- Feature selection and correlation filtering
- Missing value imputation
- Feature scaling
- K-Means clustering
- Silhouette analysis for model selection
- PCA for dimensionality reduction

## Results
- Optimal number of clusters: k = 2
- Silhouette score: 0.081
- PCA visualization shows partial separation between clusters

## Key Insight
Patients can be grouped into distinct profiles based on clinical features, although overlap is expected in medical data.

## Business / Research Value
Supports patient segmentation and exploratory analysis in healthcare datasets.
